# Play the Chaos Game

## Create beautiful fractals from random events creating chaos eventually leading to ordered structure.

### Requirements
- python3
- turtle
- argparse

### Instructions

Animation is disabled by default.

In order to run with animations do

`python3 chaos_game.py -a True`

NOTE:When animations is True the drawing process is slowed.
